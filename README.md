# Security Infrastructure Portfolio

Hands-on security analysis of infrastructure, systems, and real-world risks (including healthcare context).

---

## 🎯 Focus

This repository demonstrates practical work within:

- Infrastructure and system security  
- Network exposure and misconfigurations  
- Root cause analysis of vulnerabilities  
- Real-world impact (with focus on healthcare systems)

The goal is to show how systems fail, why it matters, and how to mitigate risks.

---

## 🧠 Approach

Each case in this repository focuses on:

1. **System Understanding**  
   What services, protocols, and components are involved?

2. **Vulnerability Identification**  
   What is misconfigured or insecure?

3. **Root Cause Analysis**  
   Why does the issue exist?

4. **Impact Assessment**  
   What are the real-world consequences?

5. **Mitigation**  
   How should it be fixed?
